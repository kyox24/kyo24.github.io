前言

时隔几个月再次捡起ML-Agents的时候发现自己有太多东西忘记了（主要是因为学的不扎实），所以想要从头记录一下学习过程。

本笔记使用的ML-Agents版本

Unity ML-Agents-release7

下面的文章建立在已经你已经拥有

  • Unity (2018.4 or later)
  • Python (3.6.1 or higher)

还需要安装·准备的东西

  • Unity Package：com.unity.ml-agents
  • Unity ML-Agents
  • Python虚拟环境
  • 3个Python Package：
    • mlagents
    • mlagents_envs
    • gym_untiy
